# Tailwinter — Журнал изменений (RU)

> Разделён по дням: подробные записи теперь лежат в `docs/changelog/YYYY-MM-DD.md`. Ниже — консолидированная копия.

## 17.12.2025
### UI / Пауза
- На ESC добавлено пауз-меню с кнопками (Настройки/Сохранить/Загрузить/Выйти) и разделом «Настройки»: галочки ограничения метели и сна дальней фауны/декора, V-Sync, лимит FPS, выбор разрешения и полноэкранного режима. Игра ставится на паузу, курсор освобождается (17.12.2025 10:00, ~25 мин).
- Исправлено закрытие пауз-меню: кнопка «Продолжить» и повторное ESC снова снимают паузу; HUD продолжает обрабатывать ввод в режиме паузы (17.12.2025 10:10, ~5 мин).

### Визуальные эффекты / Игрок
- Добавлен пар изо рта (GPUParticles3D) у головы игрока: интенсивность зависит от холода (body_temperature) и активности, плавное сглаживание, без резких скачков (17.12.2025 11:00, ~20 мин).

### UI / Эффекты
- Добавлен иней на экране (overlay): плавно нарастает при низком тепле и исчезает в тепле, усиливается от активности; слой поверх HUD (17.12.2025 11:05, ~15 мин).
- Исправлен визуал инея: увеличено разрешение текстуры, выключено повторение, включена линейная фильтрация и снижена максимальная прозрачность, чтобы убрать квадратные артефакты и жёсткий круг (17.12.2025 11:20, ~10 мин).

### Техническое
- Исправлены типы в VFX-скриптах дыхания и инея (убраны Variant-предупреждения и несовпадения типов) без изменения поведения (17.12.2025 11:10, ~5 мин).

### Геймплей / Тепло
- Костёр переведён на время горения: 1 палка = 15 с и +10°C, максимум 3 минуты и 80°C. При попытке переполнения — сообщение «Костер полон», температура отображается в названии при наведении. Температура костра линейно падает по мере сгорания (17.12.2025 11:40, ~25 мин).
- «Ощущается как» теперь учитывает: погоду (окружение), штраф ветра, бонус от костра. Игрок замерзает/греется по этой температуре: при отрицательных значениях усиливается охлаждение, при положительных — медленно согревается; бонус от костра действует только в зоне его тепла. В инвентаре по наведению на «Ощущается как» показывается разбор (погода/ветер/костер) (17.12.2025 11:50, ~20 мин).

## 16.12.2025
### Животные / Лут
- Мёртвый кролик в капкане теперь интерактивен сам по себе: после срабатывания капкана его модель остаётся на месте, подсказка по прицелу показывает `E — Поднять тушку кролика`, вес тушки рандомно 1.5–3.5 кг, при успешном подъёме предмет кладётся в инвентарь и кролик удаляется из сцены. Если инвентарь переполнен — остаётся лежать (16.12.2025 20:00, ~20 мин).

### Ресурсы / Предметы
- Палки: очистил сломанные ссылки на текстуры, модель stick.fbx теперь с тёмно-коричневым материалом и корректным именем «Палка», чтобы не было белых безтекстурных спрайтов (16.12.2025 20:10, ~10 мин).

### HUD / Инвентарь
- Починил отображение текста (кракозябры) в HUD и инвентаре: все строки переведены в UTF-8 и показаны на русском (16.12.2025 20:20, ~5 мин).

### Дебаг / Производительность
- В F3-оверлей добавлен FPS-счётчик (обновляется каждые 0.25 с), чтобы отслеживать фризы; начальная подсказка для контроля производительности. (16.12.2025 20:25, ~5 мин)

## 14.12.2025
### Мир / Погода
- Введена системная механика ветра 0–32 м/с: цели скорости/направления меняются с вероятностью каждые 10–30 секунд, шаг ограничен, сильный ветер реже пропадает мгновенно; API `world/wind.gd` (get_wind_speed_mps, get_wind_dir_xz, wind_changed) (14.12.2025 12:10, ~25 мин).
- Снеговые эффекты подчиняются ветру: направление и интенсивность хлопьев следуют ветру, метель (≥12 м/с) добавляет туман/дымку, плотность тумана растёт с усилением ветра, в ясную погоду туман ограничен (14.12.2025 12:25, ~20 мин).

### Игрок
- Вид от первого лица: HeadCamera с FOV 85, yaw/pitch без инверсии, скорости вперёд/назад/страф с понижающими коэффициентами, управление мышью и WASD, спринт с расходом выносливости (14.12.2025 12:35, ~20 мин).

### HUD / Инвентарь
- Отображение ветра в инвентаре, подсказки взаимодействия через прицел, базовый HUD для работы с новыми предметами/статусами (14.12.2025 12:55, ~15 мин).
