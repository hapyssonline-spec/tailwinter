# Tailwinter — Журнал изменений (RU)

## 14.12.2025

### Костёр / Выживание
- Добавлена механика пополнения костра деревом.
- Шкала наполненности костра отображается только в режиме дебага (F3).

### HUD / Дебаг
- В режиме дебага добавлен виджет шкалы наполненности костра; вне дебага виджет скрыт.

### Техническое
- Уточнено форматирование файла журнала; поведение игры не менялось.
- Улучшено визуальное разделение секций журнала; поведение игры не менялось.
- Уточнены правила обновления журнала для будущих задач; поведение игры не менялось.
