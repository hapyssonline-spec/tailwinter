# 2025-12-14

## Мир/Погода
- Добавлена системная модель ветра (0–75 м/с) с вероятностным обновлением цели скорости и направления, редкими порывами и ограничением шага изменения.
- Реальная скорость и направление ветра сглаживаются по времени без резких скачков, а изменения транслируются сигналом `wind_changed`.

## Игрок
- Скорость движения теперь масштабируется попутным/встречным ветром; сильный шторм даёт до ~±35% влияния без сноса при отсутствии ввода.
- В систему выживания добавлен дополнительный расход тепла от сильного ветра с плавной нелинейной зависимостью.

## Костёр
- Костёр получает плавное ускоренное затухание в зависимости от силы ветра; при штормовом ветре ресурс горения уходит за считанные секунды.

## HUD/Инвентарь
- В окне инвентаря (клавиша I) отображается текущая скорость ветра с одним знаком после запятой и обновлением по сигналу/таймеру.
